1. Overview of C Programming
THEORY EXERCISE
History and Evolution of C Programming

C programming language was developed by Dennis Ritchie in 1972 at AT&T Bell Laboratories. It was created as an improvement over the B language to develop the UNIX operating system.

Initially, programming was done using assembly language, which was machine-dependent and complex. C introduced a high-level yet system-oriented approach, making it easier to write portable and efficient programs.

Over time, C evolved with standardization:

ANSI C (C89/C90) – Standardized syntax

C99 – Added new data types and features

C11/C18 – Improved safety and performance

Importance of C Language

Fast execution speed

Low-level memory access using pointers

Portable across platforms

Foundation for languages like C++, Java, Python

Widely used in OS, embedded systems, compilers

Why C is Still Used Today

Used in operating systems (Linux, Windows kernel)

Embedded systems (microcontrollers)

Game engines

Device drivers

Real-time systems

Even/Odd Count
int a[10],e=0,o=0,se=0,so=0;
for(int i=0;i<10;i++)
{
    scanf("%d",&a[i]);
    if(a[i]%2==0){e++;se+=a[i];}
    else{o++;so+=a[i];}
}

Reverse Number
int n,rev=0;
scanf("%d",&n);
while(n>0)
{
    rev=rev*10+n%10;
    n/=10;
}
printf("%d",rev);
