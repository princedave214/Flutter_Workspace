1. Fundamental Data Types in Dart

Dart provides several built-in data types to store different kinds of values:

int

Stores whole numbers.

Example: int age = 20;

Used for counting, indexing, and numeric calculations.

double

Stores decimal numbers.

Example: double price = 99.99;

Used for measurements and precision values.

String

Stores text.

Example: String name = "Flutter";

Used for messages, names, and input/output.

bool

Stores true or false.

Example: bool isLoggedIn = true;

List

Ordered collection of values.

Example: List<int> numbers = [1, 2, 3];

Used to store multiple related values.

Set

Unordered collection with unique values.

Example: Set<int> ids = {1, 2, 3};

Map

Stores key-value pairs.

Example: Map<String, int> marks = {'Math': 90};

Used for structured data like dictionaries.

dynamic

Can store any type of value.

Example: dynamic data = "Hello";

4. Asynchronous Programming in Dart

Represents a value available later.

  return "Data Loaded";
}

async & await

Used to write asynchronous code like synchronous code.

await fetchData();

Stream

Handles multiple asynchronous values over time.

Task 2: Temperature Conversion
import 'dart:io';

double cToF(double c) => (c * 9 / 5) + 32;
double fToC(double f) => (f - 32) * 5 / 9;

void main() {
  print("1. Celsius to Fahrenheit\n2. Fahrenheit to Celsius");
  int choice = int.parse(stdin.readLineSync()!);

  print("Enter temperature:");
  double temp = double.parse(stdin.readLineSync()!);

  if (choice == 1) {
    print("Result: ${cToF(temp)} °F");
  } else {
    print("Result: ${fToC(temp)} °C");
  }
}

Task 13: Sort Without Built-in
void bubbleSort(List<int> list) {
  for (int i = 0; i < list.length; i++) {
    for (int j = 0; j < list.length - 1; j++) {
      if (list[j] > list[j + 1]) {
        int t = list[j];
        list[j] = list[j + 1];
        list[j + 1] = t;
      }
    }
  }
}
